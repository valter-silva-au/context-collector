# Context Collector

A browser extension that extracts conversations, emails, messages, and posts from your logged-in web sessions (WhatsApp, Telegram, Gmail, LinkedIn, etc.) and exports them as structured Markdown files for AI agent context bootstrapping.

## Why

AI coding agents work better with personal context. This tool lets you download your conversations, emails, and social interactions as organized Markdown files that any AI agent can consume:

```
extracts/
  gmail/emails/2026-03-17-project-update.md
  whatsapp/chats/dad.md
  whatsapp/chats/work-team.md
  telegram/chats/dev-community.md
  linkedin/posts/saved-posts.md
  linkedin/messages/recruiter-thread.md
```
